📝 Introduction
Welcome to the Iterator and Function Object Data Processing lab! In this hands-on activity, you'll build a data processing system that demonstrates practical usage of different iterator types and custom function objects with real data processing scenarios.

In professional software development, understanding how to choose and use appropriate iterators is crucial for writing efficient, maintainable code. Different containers provide different iterator capabilities, and knowing when to use bidirectional vs. random access iterators can significantly impact performance.

Your system will process numerical data using various iterator approaches and apply custom transformations through function objects, focusing on the iterator categories and function object concepts you've learned in this module.

For this lab, you'll need:
C++ compiler supporting C++17 or later
STL headers (<algorithm>, <iterator>, <numeric>)
Understanding of iterator categories from the readings
